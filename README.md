#Netflix Clone:

A responsive Netflix Clone built using React.js, HTML, CSS, and JavaScript, which fetches real movie data using the OMDb API.
This project replicates the core UI and functionality of Netflix for learning and practice purposes.

#Features:

 Netflix-like UI & layout

 Search movies using OMDb API

 Movie cards with poster, title & year

 Play trailer button (YouTube integration)

 Fully responsive design

 Reusable React components

 Fast & optimized frontend


 #API Used:

OMDb API
Fetches real-time movie data like title, poster, year, and ratings.
